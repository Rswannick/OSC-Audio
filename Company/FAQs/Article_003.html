How do I get my Factory Presets for sumFX iOS?

The Factory sumFX Presets can be found at the link below. Download these on your iOS device, then double tap the .zip file icon to extract the files and move them to the appropriate location.

We suggest storing them in the iOS location below. This will help avoid any permissions issues on iOS. On Mac/PC presets can be stored anywhere you'd like.

DOWNLOAD FACTORY sumFX PRESETS

Please note, sumFX presets are designed to be crossplatform loaded on iOS or Mac/PC. To get started transfering your presets, first find their location via a Finder (Mac), File Explorer (Windows), or Files App instance on iOS.

*Sub folders can be used on all platforms. On iOS, be sure to store this in the App's folder as mentioned above.

Mac: Mac HD -> Users -> Shared -> Shared Documents -> OSC Audio
PC: C:// -> Users -> Shared -> Public Documents -> OSC Audio
iOS: Files App -> On my iPhone/iPad -> sumFX
